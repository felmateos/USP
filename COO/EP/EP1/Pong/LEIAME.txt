As seguintes linhas de comando são necessárias para compilar o programa no cmd do Windows:

cd "caminho do arquivo src aqui"
javac *.java
java Pong



Caso o programa não execute, utilize as seuguintes linhas de comando:

cd "caminho do arquivo src aqui"
javac *.java
java Pong 10 true
java Pong
